# 디지털 시계 | 22.10.21 ~ 25

URL : https://chunjaeilu.github.io/digi_clock/

- 개인과제 :: Javascript 활용한 디지털시계 만들기

- 과제내용
  - new Date 활용, 시간정보 불러오기
  - 시, 분, 초 표시
  - 12시간제, am/pm 표시, 10의자리 0 표시
  - 기타사항 자유, 나만의 시계 만들기

==============================================

- 심플한 스마트워치를 형상화 한 웹시계
- 1분 마다 테두리 라인이 조금씩 채워지고 매 시 정각에 초기화
- 우측상단 버튼을 클릭하면 테마컬러를 바꿀 수 있음
